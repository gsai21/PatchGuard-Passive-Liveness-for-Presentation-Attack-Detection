# PatchGuard-Passive-Liveness-for-Presentation-Attack-Detection
Designed and trained a custom 5M param MiniViT for presentation attack detection on SynthASpoof Dataset.

# Steps to follow
1. Arrange in this foramt
  your_project/
    ├─ app.py
    └─ static/
       ├─ index.html
       └─ main.js

2. Run this command in terminal of that folder
   uvicorn app:app --host 0.0.0.0 --port 8000 --reload

3. Open this link in your browser
   http://localhost:8000/
